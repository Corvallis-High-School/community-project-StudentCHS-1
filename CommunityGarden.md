# Community Garden
This will be an application to keep track of member data for a community garden
## Specifications
- The application will store information about each member, including their name, the bed they are assigned to and their outstanding dues
- The application will allow adding new members, updating existing member information, and removing members from the system
- The application will provide a way to display all member information in a readable format
- The application will include functionality to calculate and display the total outstanding dues for all members
- The application will include a user interface to interact with the system, allowing users to perform all of the above actions easily
